Logical OR Search
=================================================================================================================================

Description:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Logical OR search is supported either by typing "OR" or using a pipe "|".
Search results will contain ANY of the keywords used to search.
