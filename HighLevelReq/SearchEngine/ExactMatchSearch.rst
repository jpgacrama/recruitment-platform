.. _ExactMatchSearch:

Exact Match Search
=================================================================================================================================

Description:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Searching for exact match is done by placing the search string in quotation marks.
E.g. "I want this exact sentence" returns results which shows that exact sentence including its case.
